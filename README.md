# Analise Qualidade do ar
Projeto de Análise de dados com informações sobre a qualidade do ar.

Foi realizado a importação dos dados, análise exploratória, tratamento de dados discrepantes, correlações, gráficos e  levantamento de insights. Ao final do projeto, foi feito uma conclusão final com os insights principais do projeto

Foi criado no Google Colab com Python e suas bibliotecas de análise de dados: Pandas, Matplotlib e Numpy
